# Neurotic Manual

Neurotic is a 10‑algorithm neural‑inspired processor/synth for the kxmx_bluemchen hardware. Each algorithm uses four controls (C1–C4) mapped to the two knobs/CVs plus two menu parameters.

## Controls
- **Knob 1 + CV1 (C1)**: Primary algorithm parameter (performance control).
- **Knob 2 + CV2 (C2)**: Secondary algorithm parameter (performance control).
- **Menu C3/C4**: Algorithm-specific parameters.
- **MIX**: Dry/wet balance.
- **ALG**: Algorithm index (0–9).
- **OUT**: Output trim.

## Menu System
Follows `docs/menu-system.md`.
- **Master** page: MIX, ALG, OUT
- **Algo** page: C3, C4 (labels change per algorithm)

## Algorithms
0. **NCR**: Neural Cross‑Resonator
1. **LSB**: Latent Spectral Braider
2. **NTH**: Neural Tape Hydraulics
3. **BGM**: Binaural Gesture Mapper
4. **NFF**: Neural Formant Forge
5. **NDM**: Neural Diffusion Multiband
6. **NES**: Neural Energy Shaper
7. **NHC**: Neural Harmonic Cartographer
8. **NPL**: Neural Phase Loom
9. **NMG**: Neural Micro‑Granulator
